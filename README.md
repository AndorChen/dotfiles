#Andor’s dotfiles

##安装

```bash
git clone https://AndorChen@github.com/AndorChen/dotfiles.git
cd dotfiles
rake install
```

新安装的 Mac，可以执行以下命令设置一些系统级的设定：

```bash
osx_defaults.sh
```
